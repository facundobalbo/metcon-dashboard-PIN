# METCON S.A.S — Dashboard Operativo

Dashboard en tiempo real para gestión de maquinaria en obra. Lee datos automáticamente desde Google Sheets.

**[Ver Dashboard →](https://facundobalbo.github.io/metcon-dashboard-PIN/)**

## Qué muestra

- Horas máquina acumuladas por equipo y operador
- Roturas, paradas y motivos (rotura, combustible, tiempo muerto)
- % de disponibilidad y productividad por máquina
- Alertas automáticas (datos desactualizados, roturas frecuentes, falta de combustible)
- Detalle por máquina con historial de fallas

## Stack

HTML + CSS + JS vanilla. Sin dependencias, sin build. Un solo archivo.

Los datos se leen desde una Google Sheet publicada como CSV. Se auto-refresca cada 5 minutos.

## Obra actual

Cierre Perimetral — Parque Industrial Norte, San Luis.
